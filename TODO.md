## Setup Instructions

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Configure API keys in config.json:
   - Groq API key for AI conversations
   - OpenWeatherMap API key for weather
   - News API key for news

3. Run Bosco Core:
   ```bash
   python main.py
   ```
